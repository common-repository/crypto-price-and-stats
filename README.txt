Crypto Price And Stats is a WordPress plugin displays live prices and stats of crypto coins.

== Description ==

Crypto Price And Stats WordPress plugin displays live prices and stats of cryptocurrency coins.  It is easy to use and you can create shortcode and widgets in few steps.

Crypto Price And Stats plugin has 8 different styles/templates for showing live price and stats of cryptocurrencies to your website. This widget will help you add useful information to your crypto related website. Display crypto prices, convert and compare them in beautiful way with the help of this plugin.

= How to use plugin? =

Downloade & install the plugin into your website. After successful installation of the plugin, go to plugin's "DOC" menu for help and instructions.

== Installation ==

Install Crypto Price And Stats either via the WordPress.org plugin directory or by uploading the files directly to your server.

After activating Crypto Price And Stats plugin, go to plugin menu.

Login into WordPress admin dashboard. Go to menu: Crypto Stats

Choose Template Crypto Stats & configure settings and save.

Copy shortcode and paste shortcode into any Page / Post. And view page for Crypto Price And Stats output.

That's it. You're ready to go!
